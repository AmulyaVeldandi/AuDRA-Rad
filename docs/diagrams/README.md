# Architecture Diagrams

This directory contains the draw.io source file for all AuDRA-Rad architecture diagrams.

## File

**architecture.drawio** - All diagrams in one file with 4 pages:
  - Page 1: System Context (C4 Level 1)
  - Page 2: Container Architecture (C4 Level 2)
  - Page 3: Report Processing Sequence
  - Page 4: AWS Deployment Architecture

## Editing Diagrams

### Online (Recommended)
1. Visit https://app.diagrams.net/
2. Click "Open Existing Diagram"
3. Upload `architecture.drawio`
4. Edit and save

### Desktop App
1. Download draw.io desktop from https://github.com/jgraph/drawio-desktop/releases
2. Open `architecture.drawio`
3. Edit and save

### VS Code Extension
1. Install "Draw.io Integration" extension
2. Open `architecture.drawio`
3. Edit inline

## Exporting

### From draw.io (online or desktop):
- **PNG:** File → Export as → PNG (for presentations)
- **SVG:** File → Export as → SVG (for documentation, scalable)
- **PDF:** File → Export as → PDF (for printing)

### Export Options:
- Select specific page or all pages
- Adjust zoom level (100% recommended)
- Include diagram name in filename
